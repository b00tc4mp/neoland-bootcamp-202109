# Terminal commands

- pwd (path to working directory)

```sh
% pwd
/Users/Igluit3@MacBook-Air
```

- ls (list files & folders)

```sh
% ls
Applications
Music
Creative Cloud Files
Desktop
```

- cd (change directory)

````sh
% cd bootcamp

- touch (to create a file from scratch)

-mkdir (make directory
)

```sh
% mkdir bootcamp
````

- mv (move/rename file or folder)

```sh
% mv cli cli-demo
```
