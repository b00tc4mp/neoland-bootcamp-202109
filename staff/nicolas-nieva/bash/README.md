# Bash commands 

- pwd (path to working directory)

```sh
$ pwd 
/Users/geometry/bootcamp/cli-demo
```sh

- ls (list files & folders)

- cd (change directory)
```sh
$ cd - cli-demo
```
   
-  cd .. (come back to parent)
```
$ cd ..
```

-  cd (change to user directory) 
```
$ cd 
```

- touch (create file from scratch)

```sh
$ touch README.md
```

- move (rename)
```sh
$ move cli cli-demo
```
- rm (remove)
```sh
$ rm README.md
```
