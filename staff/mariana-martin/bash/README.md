# Bash commands

- pwd (path to working directory)

```sh
$ pwd
/Users/mariana
````

- ls (list files & folders

```sh
$ ls
```

- cd ..

- cd (change directory)
```

cd .. (change to parent directory)


-touch (to create a file from scratch)
```sh
touch README.md
```

- mkdir (make directory)

```sh
mkdir dir nombre
```

- mv (move file or folder)

```sh
$ mv cli  cli-demo
```

- rm (remove file)
```sh
$ rm README.md
````

- clear (clear screen = CTRL + L)

```sh
$ clear
```
