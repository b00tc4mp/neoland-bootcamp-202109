- Para instalar React
    `npm install -g create-react-app`

- Para crear un nuevo proyecto React ejecutamos
    `react-create-app name-app` `npx react-create-app name-app`

- Para instalar las dependencias de un proyecto (node_modules)
    `cd name-app`   `npm install`   `npm ci`

- Para lanzar nuestro servidor de desarrollo
    `npm start` `npm run start`

- para crear los archivos comprimidos en /dist
    `npm run build`
